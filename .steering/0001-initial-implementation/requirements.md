# Requirements (0001 Initial Implementation)

## 1. 目的
PoCとして「肌荒れが気になる夜に、家にあるものだけで最短のケア指示を返す」体験の価値を検証する。

## 2. 変更・追加する機能
- 30秒問診（症状＋強度）
- 手持ちコスメ登録（商品名＋カテゴリ）
- 今夜の指示（使う/休む/手順/様子見ルール）
- 翌日フォロー（3択＋任意メモ）
- 注意事項と受診誘導の表示

## 3. ユーザーストーリー
1) 肌荒れが気になる夜、短時間で今夜のケア指示が欲しい
2) 家にあるものだけで安心できるケアがしたい
3) 翌日に状態を振り返って判断を調整したい

## 4. 受け入れ条件
- 問診が60秒以内に完了できる
- 商品名＋カテゴリで手持ち登録できる
- 回答が固定フォーマットで表示される
- 医療行為ではない注意書きが表示される
- 危険サインは受診誘導を出す
- データは匿名・端末内のみで保存される

## 5. 制約事項
- 外部APIは使用しない
- 成分解析/相性チェックは対象外
- 医療判断に踏み込む表現はしない
