# Glossary

## 1. ドメイン用語
- 肌荒れ: 赤み/ヒリつき/乾燥/ニキビっぽい/かゆみ等を含む肌の不調
- 問診: 症状と強度を短時間で確認するフロー
- 今夜の指示: 当日のケアを最小限の手順で示す提案
- 受診誘導: 危険サインがある場合に医療機関の受診を促す表示
- 手持ちコスメ: ユーザーが家に所有しているスキンケア製品

## 2. ビジネス用語
- PoC: 概念実証。価値があるかを短期で検証する取り組み
- KPI: 目標達成度を測る指標
- 継続率: 一定期間内に再利用した割合

## 3. UI/UX用語
- 30秒問診: 1分以内に終わる短い質問フロー
- フォローアップ: 翌日の状態確認（良くなった/変わらない/悪化）
- 注意事項: 医療行為ではないことを伝える表示
